# 熟睡ベンチマーク | Satellite Site (Lighthouse)

熟睡ベンチマークとは？
「熟睡ベンチマーク」は寝室環境スコアリングツールです。

01
寝苦しさを「見える化」
室温や湿度だけでなく、住宅の断熱性能や季節ごとの気候データを元に、あなたの寝室のコンディションを独自にスコア化します。

02
デバイス運用の最適解
エアコンの「除湿」と「冷房」の使い分けや、サーキュレーターの最適な配置場所など、今すぐ実践できる具体的なアクションをサポート。

03
温度と湿度に計算ロジックを工夫して、UIと物理の整合性のバランスを取りました。つまり、厳密な計測器ではありません。
