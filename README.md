# 熟睡ベンチマーク | Satellite Site (Lighthouse)

熟睡ベンチマークとは？
「熟睡ベンチマーク」は寝室環境スコアリングツールです。寝苦しさや不眠症の啓蒙サイトです。

01
寝苦しさを「見える化」
室温や湿度だけでなく、住宅の断熱性能や季節ごとの気候データを元に、寝室のコンディションを独自にスコア化します。人間側の計測はバラツキがあり過ぎるので、ツールの機能は環境に特化しました。

02
デバイス運用の最適解
エアコンの「除湿」と「冷房」の使い分けや、サーキュレーターの最適な配置場所など、今すぐ実践できる具体的なアクションをサポートします。

03

トップページのツールには温度と湿度の計算ロジックを工夫して、UIと物理の整合性のバランスを取りました。GEMINI 3とClaude Opus 4.5の二重チェックで作成。また厳密な計測器ではありません。あくまで目安を出す可視化UXです。
